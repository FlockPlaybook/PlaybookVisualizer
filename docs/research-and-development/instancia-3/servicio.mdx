# Brinda un servicio de excelencia

Actua ante las necesidades y pedidos de un cliente, interactuando si fuera necesario, para responder dudas o consultas

**Ejemplos:**
- Interactúa con el cliente contestando dudas e inquietudes de manera efectiva y profesional.
- Realiza el onboarding técnico de los nuevos miembros del equipo del proyecto donde trabaja".
