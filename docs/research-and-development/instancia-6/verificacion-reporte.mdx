# Validación, métricas y evidencia

Establece los marcos de validación y los criterios de evidencia que rigen a varias líneas de investigación, incluyendo métodos avanzados y estándares de calidad. Determina qué nivel de evidencia es suficiente para decidir inversión, escalado, transferencia o cierre de una línea.

**Ejemplos:**

- Define el estándar de 'evidencia suficiente' que habilita una decisión de inversión.
- Aprueba qué métodos de medición se consideran válidos para toda la línea.
- Define cómo se audita la calidad de la evidencia entre distintas iniciativas.
- Decide qué nivel de evidencia se exige antes de escalar o cerrar una línea.
