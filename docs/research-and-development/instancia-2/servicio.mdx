# Brinda un servicio de excelencia

Cultiva relaciones sólidas y de confianza con el cliente interno y otros equipos, manteniendo una comunicación abierta y colaborativa para entender mejor sus necesidades y expectativas.

**Ejemplos:**
-  Comunica de manera clara y efectiva con el cliente interno y otros equipos, ofreciendo asesoramiento técnico y orientación cuando sea necesario.
- Colabora estrechamente con otros miembros del equipo para asegurar la alineación de objetivos y la comprensión de las necesidades del cliente interno.
