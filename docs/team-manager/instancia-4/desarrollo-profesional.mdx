# Desarrollo profesional

Establece estándar de excelencia en desarrollo profesional. Identifica proactivamente expectativas de desarrollo. Genera sinergia y aprendizaje continuo.

**Ejemplos:**
- Establecer el estándar de excelencia en gestión del desarrollo profesional del equipo.
- Identificar proactivamente las expectativas de desarrollo de cada colaborador.
- Generar sinergia entre equipos para maximizar el aprendizaje colectivo.
- Diseñar planes de desarrollo personalizados para miembros clave del equipo.
