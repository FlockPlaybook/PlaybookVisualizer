# Gestión de proyecto

Colabora en la planificación y priorización de funcionalidades. Gestiona carga de trabajo y capacidad del equipo. Identifica riesgos como requerimientos mal definidos.

**Ejemplos:**
- Planificar sprints considerando la capacidad real del equipo.
- Gestionar la carga de trabajo del equipo para evitar sobrecargas.
- Identificar y escalar requerimientos mal definidos antes de que impacten el sprint.
- Colaborar en la priorización del backlog con el cliente y el FA.
