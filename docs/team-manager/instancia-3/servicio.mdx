# Brinda un servicio de excelencia

Facilita espacios de colaboración. Realiza evaluaciones periódicas de procesos. Propicia encuentros con BM para compartir el estado del proyecto.

**Ejemplos:**
- Facilitar workshops y espacios de colaboración con el cliente.
- Realizar evaluaciones periódicas del proceso de trabajo y proponer mejoras.
- Organizar encuentros regulares con el Business Manager para informar el estado del proyecto.
