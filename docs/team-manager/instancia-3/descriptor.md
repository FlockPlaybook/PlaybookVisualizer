# Team Manager - Instancia 3

## Descriptor

| Dimensión | Descripción |
|---|---|
| **Complejidad** | Complejidad media/alta. Interacciones regulares con cliente. No requiere instrucción en tareas rutinarias. |
| **Autonomía** | Alta autonomía en la gestión. Lidera proyectos complejos de forma independiente. |
| **Impacto de las decisiones** | Las decisiones impactan en el equipo, el cliente y el proyecto en su totalidad. |
| **Alcance** | Lidera proyectos complejos. Referente de gestión dentro del área. |
