# Metodología y mejora continua

Evalúa y ajusta el proceso ágil. Resuelve impedimentos complejos. Introduce prácticas adicionales de metodologías ágiles. Examina métricas complejas para identificar tendencias.

**Ejemplos:**
- Evaluar y ajustar el proceso ágil del equipo basándose en retrospectivas y métricas.
- Resolver impedimentos complejos de forma autónoma.
- Introducir prácticas de XP, Lean o Design Thinking según el contexto.
- Analizar métricas de flujo (lead time, cycle time, WIP) e identificar tendencias.
