# Comunicación con stakeholders

Establece y mantiene una comunicación efectiva con los stakeholders internos, incluyendo al equipo de desarrollo, y comienza a interactuar con clientes externos. Recopila requisitos y expectativas de manera proactiva y aprende a traducirlos para el equipo. Comienza a preparar informes y actualizaciones de proyecto básicos.

**Ejemplos:**
-  comienza a interactuar con clientes externos, asegurándose de comprender sus necesidades y expectativas.
- Colabora estrechamente con el equipo de desarrollo, facilitando la comunicación bidireccional para asegurar una comprensión clara de los requisitos del proyecto y transmitiendo eficientemente la información del equipo a los stakeholders.
-Traduce de manera efectiva los requisitos y expectativas recopilados para que sean comprensibles y ejecutables por el equipo de desarrollo
- Facilita reuniones efectivas con stakeholders, asegurando una participación significativa y una comprensión clara de los temas discutidos.
