# Team Manager - Instancia 5

## Descriptor

| Dimensión | Descripción |
|---|---|
| **Complejidad** | Problemas moderadamente complejos. Crea nuevas soluciones. Interacción con niveles gerenciales. |
| **Autonomía** | Máxima autonomía. Mínima guía requerida. Interacción directa con la dirección. |
| **Impacto de las decisiones** | Las decisiones impactan en el área y en la organización. |
| **Alcance** | Referente de gestión para el área. Lidera iniciativas organizacionales. |
