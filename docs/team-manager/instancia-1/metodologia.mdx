# Metodología y mejora continua

Facilita ceremonias ágiles básicas. Aplica principios fundamentales del agilismo. Implementa métricas básicas de seguimiento.

**Ejemplos:**
- Facilitar daily standups, sprint plannings y retrospectivas con eficacia.
- Aplicar principios ágiles (Scrum, Kanban) en el contexto del equipo.
- Implementar métricas básicas como velocidad del equipo y ciclo de trabajo.
- Documentar el proceso del equipo y mantenerlo actualizado.
