# Liderazgo de equipo

Fomenta un ambiente colaborativo. Demuestra escucha activa. Aborda conflictos constructivamente.

**Ejemplos:**
- Crear un ambiente de trabajo psicológicamente seguro para el equipo.
- Practicar escucha activa en todas las interacciones con el equipo.
- Abordar conflictos entre miembros del equipo de forma constructiva.
- Celebrar los logros del equipo y reconocer contribuciones individuales.
