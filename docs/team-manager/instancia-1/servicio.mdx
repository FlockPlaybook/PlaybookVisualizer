# Brinda un servicio de excelencia

Orienta acciones al cumplimiento de objetivos grupales. Garantiza acceso a herramientas y recursos. Establece protocolo de comunicación interna.

**Ejemplos:**
- Asegurar que el equipo tenga acceso a todas las herramientas y recursos necesarios.
- Establecer protocolos de comunicación interna claros y eficientes.
- Orientar las acciones del equipo hacia el cumplimiento de objetivos grupales.
