# Comunicación con stakeholders

Se comunica de forma exitosa con stakeholders y facilita la transmisión de información relevante con los miembros del equipo y los diferentes interlocutores.

Fomenta un ambiente de apertura y disposición para la comunicación.

Transmite información de manera clara, precisa y a tiempo.

**Ejemplos:**
- Muestra habilidades de escucha activa.
- Valida la información recibida para asegurar una buena comunicación.
