# Comunicación con stakeholders

Inicia la comunicación con los stakeholders, estableciendo una conexión inicial con los miembros del equipo y clientes internos. Fomenta un ambiente de apertura y disposición para la comunicación, mostrando habilidades básicas de escucha y transmitiendo información de manera clara y directa. Se centra en establecer una base sólida para futuras interacciones y construir relaciones iniciales con los stakeholders internos.
