# Calidad y validación

Participa como observador/a de las PDU, trabaja en conjunto otros/as diseñadores/ras de mayor expertise para su armando. 

Conoce algunos parámetros de accesibilidad, pero aún carece del nivel de profundidad técnica necesario para su correcta implementación.
 
Comienza a tomar nocion sobre la importancia de las métricas dentro de un proyecto, pero aún no tiene el conocimiento de cómo utilizarlas

**Ejemplos:**
- Realiza perfiles de proto persona en base a un input o como disparadores de hipótesis.
- Realiza benchmark de productos análogos o similares.
- Tiene un conocimiento general de las normas de accesibilidad, y puede discernir entre criterios A, AA, AAA
