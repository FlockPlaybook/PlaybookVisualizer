# Diseño y creación de prototipos

Diseña interfaces de baja complejidad con el acompañamiento de otro/a diseñador/a de mayor expertise.
 
Es capaz de generar prototipados interactivos mediante las herramientas pertinentes. 

Cuenta con el conocimiento de patrones de diseño a utilizar pero todavía requiere seguimiento.

**Ejemplos:**
- Generar entregables con supervisión
- Analiza otros proyectos y productos para entender patrones de usabilidad.
- Trabaja con styleguides y componentes provistos, creando diseños elementales para el producto.
