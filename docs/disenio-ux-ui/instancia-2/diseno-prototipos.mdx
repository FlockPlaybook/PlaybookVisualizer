# Diseño y creación de prototipos

Diseña interfaces de baja complejidad y genera prototipados interactivos. Requiere seguimiento para asegurar calidad y consistencia.

**Ejemplos:**
- Diseñar pantallas sencillas siguiendo el Design System del proyecto.
- Generar prototipos interactivos de baja a media fidelidad en Figma.
- Iterar sobre los diseños incorporando feedback del equipo.
- Documentar componentes y variaciones utilizadas en cada entrega.
- Comunicar avances al referente y ajustar según indicaciones.
