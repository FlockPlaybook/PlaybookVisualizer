# Brinda un servicio de excelencia

Escucha atentamente las solicitudes del cliente interno, respondiendo de manera efectiva y tomando acciones concretas para dar forma a soluciones creativas. Enfoca sus esfuerzos en generar resultados visuales y estéticos que cumplan con las expectativas del cliente.

**Ejemplos:**
- Colabora activamente con el cliente interno y externo, participando en sesiones de feedback y reuniones para comprender a fondo las expectativas, necesidades y preferencias en cuanto al diseño web.
- Busca apoyo y orientación de referentes o expertos en diseño web en situaciones desafiantes o en casos novedosos, asegurando que las propuestas y soluciones sean innovadoras y estén alineadas con las expectativas del cliente.
