# Estrategia de producto

Colabora en el aterrizaje de casos de uso. Comprende la estructura del producto y el flujo de información entre pantallas.

**Ejemplos:**
- Colaborar con el FA o PO en la traducción de requerimientos en casos de uso visuales.
- Documentar flujos de usuario (user flows) para pantallas específicas.
- Comprender el mapa de pantallas del producto y su jerarquía de información.
- Participar en reuniones de refinamiento aportando perspectiva visual.
