# Estrategia de producto

Colabora en el aterrizaje de los casos de uso.

Participa como observador/a de las instancias de refinamiento del producto.

**Ejemplos:**
- Comprende, en un alto nivel, el scope del producto en el que trabaja.
- Entiende las estructuras de un producto y puede organizar los contenidos entre navegación principal, secciones o páginas.
