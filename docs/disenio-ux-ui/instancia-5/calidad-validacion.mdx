# Calidad y validación

Lidera y gestiona el proceso completo de PDU. Colabora en la construcción de templates y procesos de validación escalables para el equipo.

**Ejemplos:**
- Liderar end-to-end el proceso de PDU: planificación, ejecución y análisis.
- Construir templates reutilizables para pruebas de usabilidad y evaluaciones heurísticas.
- Establecer métricas de calidad de diseño para el equipo.
- Formar a perfiles junior en metodologías de validación de diseño.
- Generar reportes ejecutivos sobre el estado de calidad del producto.
