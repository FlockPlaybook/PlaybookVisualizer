# Diseño y creación de prototipos

Crea interfaces de media o alta complejidad para los productos tecnológicos desarrollados, velando por la calidad acorde a los estándares y buenas prácticas. 

Conoce el correcto uso de los patrones de diseño para poder crear una experiencia coherente e intuitiva. 

Propone soluciones gráficas que elevan el estándar de calidad de producción del equipo.

**Ejemplos:**
- Diseña los prototipos considerando los múltiples recursos (fotografía, contenido, ilustración, motion) logrando la mayor fidelidad del prototipo final.
- Crea nuevos styleguides de acuerdo a los templates.
- En términos de arquitectura de información es cuidadoso con el correcto uso del lenguaje (wording)
- Genera hipótesis a partir de benchmark.
