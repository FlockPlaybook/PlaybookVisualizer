# Estrategia de producto

Documenta casos de uso de forma completa y autónoma. Realiza estimaciones de tiempo para tareas de diseño y las comunica al equipo.

**Ejemplos:**
- Elaborar documentación completa de casos de uso con criterios de aceptación visuales.
- Estimar esfuerzo de diseño para features y comunicarlo al equipo de gestión.
- Proponer mejoras al proceso de definición de producto en base a experiencias previas.
- Colaborar con el equipo de producto en la construcción del roadmap visual.
