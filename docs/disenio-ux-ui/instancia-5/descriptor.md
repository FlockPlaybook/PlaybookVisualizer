# UX/UI - Instancia 5

## Descriptor

| Dimensión | Descripción |
|---|---|
| **Complejidad** | Problemas moderadamente complejos. Guía al equipo al definir métodos. |
| **Autonomía** | Resuelve problemas complejos con mínima guía. Interacción con management. |
| **Impacto de las decisiones** | Las decisiones impactan en el equipo de diseño y en el producto. |
| **Alcance** | Referente técnico dentro del equipo. Contribuye al desarrollo del talento. |
