# Calidad y validación

Manejo fluido en PDU comparativas o sucesivas. Realiza informes de accesibilidad de alto nivel. Cuestiona la objetividad de los datos y propone metodologías de validación más robustas.

**Ejemplos:**
- Diseñar y ejecutar programas completos de investigación de usuarios.
- Cuestionar metodologías de validación existentes y proponer mejoras.
- Elaborar informes ejecutivos de accesibilidad y calidad de diseño.
- Definir estándares de calidad de diseño para toda la organización.
- Generar aprendizajes transversales a partir de PDU y compartirlos formalmente.
