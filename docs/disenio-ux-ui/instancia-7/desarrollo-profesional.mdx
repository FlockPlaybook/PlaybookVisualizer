# Desarrollo profesional

Comprende y domina un amplio espectro de disciplinas, puede relacionarlas y tomar decisiones. 

Es responsable de la guía y formación de los/las referentes técnicos del equipo. 

Diferencia criteriosamente la aplicación de qué metodología o herramienta es más pertinente usar para promover el desarrollo fluido de una actividad. 

Es capaz de llevar adelante capacitaciones internas como externas, teniendo un contacto fluido tanto a compañeros del área como stakeholders.

**Ejemplos:**
- Genera material de comunicación para el área, interna y externamente.
- Motiva e impulsa el desarrollo de conocimiento en sus pares a partir del feedback que brinda.
- Genera la documentación necesaria para que los procesos de trabajo de lleven adelante.
- Genera input técnico para PDMs y Heads para las capacitaciones del equipo.
- Es responsable junto con PDMs y TLs de la actualización del proceso de capacitaciónes técnicas y trainings de nuevos ingresos y del equipo.
