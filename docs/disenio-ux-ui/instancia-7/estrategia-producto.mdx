# Estrategia de producto

Partner estratégico en la fijación de objetivos de producto. Analiza contenidos para optimización del SEO desde la perspectiva de diseño. Ayuda a generar el roadmap de largo plazo.

**Ejemplos:**
- Participar en la definición de OKRs de producto con perspectiva de diseño.
- Analizar estructuras de contenido y proponer mejoras para SEO y UX.
- Colaborar con Heads de producto en la visión de largo plazo del producto.
- Liderar iniciativas de innovación de producto apoyadas en investigación de usuarios.
