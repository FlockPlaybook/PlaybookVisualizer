# Diseño y creación de prototipos

Se incorpora para destrabar casos puntuales de alta complejidad. Propone y ejecuta mejoras estructurales a procesos de diseño. Detecta errores sistémicos y elabora planes de acción.

**Ejemplos:**
- Intervenir en proyectos críticos para resolver bloqueos de diseño de alta complejidad.
- Proponer cambios en el proceso de diseño que impacten en toda el área.
- Elaborar planes de acción ante inconsistencias detectadas en el producto.
- Definir estándares de diseño y asegurar su adopción en toda la organización.
- Generar input técnico de diseño para decisiones de producto de largo plazo.
