# Diseño y creación de prototipos

Crea interfaces con creciente independencia, ejecutando tareas complejas con mínima ayuda externa. Propone soluciones de diseño ante nuevos requerimientos.

**Ejemplos:**
- Diseñar módulos completos del producto de forma autónoma.
- Proponer variantes de diseño con fundamentos visuales y de UX.
- Coordinar con el equipo de desarrollo para validar viabilidad técnica.
- Crear y mantener componentes en el Design System del proyecto.
- Gestionar el propio flujo de trabajo y respetar timings de entrega.
