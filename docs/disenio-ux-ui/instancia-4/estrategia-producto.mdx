# Estrategia de producto

Lleva adelante el aterrizaje de casos de uso con mayor autonomía. Co-crea junto al cliente interno o externo. Consulta herramientas de análisis para fundamentar decisiones.

**Ejemplos:**
- Liderar workshops de diseño con el cliente para definir flujos clave.
- Consultar herramientas de análisis (Hotjar, Mixpanel, GA) para tomar decisiones.
- Proponer nuevos casos de uso basados en datos de uso real del producto.
- Co-crear con el cliente prototipos de media/alta fidelidad para validar ideas.
