# Brinda un servicio de excelencia

Es responsable de abordar los pedidos del cliente interno y acude de manera oportuna a su referente cuando se requiere tomar decisiones fuera de su alcance.

**Ejemplos:**
- Responde a preguntas relacionadas con el diseño, la estética y la calidad de sus tareas actuales o anteriores.
- Mantiene informado al cliente interno (por ejemplo, referente de proyecto) acerca de los progresos en las tareas de diseño web que está llevando a cabo.
