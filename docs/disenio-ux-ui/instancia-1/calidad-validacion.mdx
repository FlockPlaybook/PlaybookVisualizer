# Calidad y validación

Participa como observador en Pruebas de Usabilidad (PDU). Se interioriza sobre conceptos de accesibilidad. Participa en ejercicios de Benchmarking.

**Ejemplos:**
- Asistir como observador a sesiones de prueba de usabilidad.
- Leer documentación sobre accesibilidad web (WCAG) y comentarla con el equipo.
- Participar en ejercicios de benchmarking analizando productos competidores.
- Registrar observaciones durante las sesiones de testing para compartir con el referente.
