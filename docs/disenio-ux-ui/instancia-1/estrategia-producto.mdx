# Estrategia de producto

Participa como oyente en procesos de priorización de producto. Comprende el porqué del proceso de diseño y su rol dentro del ciclo de producto.

**Ejemplos:**
- Asistir a reuniones de priorización de producto como observador.
- Leer y comprender la documentación de producto y los objetivos del proyecto.
- Familiarizarse con el proceso de diseño y su impacto en el desarrollo.
- Realizar preguntas a referentes para entender las decisiones de producto.
