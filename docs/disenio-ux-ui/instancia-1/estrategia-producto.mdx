# Estrategia de producto

Participa como oyente en los procesos de priorización de los flujos a trabajar. 

Suele tener un alcance limitado de acción sobre las necesidades del producto y objetivos del proyecto.

**Ejemplos:**
- Comprende el por qué del proceso de diseño, y se alinea a la manera definida previamente sobre cómo ejecutarlo.
- Indaga con su equipo el proceso de refinamiento de tareas y las necesidades del proyecto.
