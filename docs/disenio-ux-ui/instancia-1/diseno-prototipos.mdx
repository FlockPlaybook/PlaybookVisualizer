# Diseño y creación de prototipos

Aprende sobre patrones de diseño pertinentes a la disciplina para poder aplicarlos correctamente. 

Comienza a entender como diseñar wireframes en baja calidad a partir de una solicitud específica de otro/a diseñador/a de mayor seniority.

**Ejemplos:**
- Se familiariza sobre el uso de herramienta de diseño y el proceso de trabajo
- Genera entregables con supervisión frecuente.
- Buscan asesoría técnica de otros/as diseñadores/as del equipo.
- Se interioriza en los procesos de delivery, y colabora con los pedidos de sus referentes en la preparación del mismo.
