# Brinda un servicio de excelencia

Brinda soporte especializado de diseño al equipo. Participa en iniciativas de mejora continua del área.

**Ejemplos:**
- Resolver consultas de diseño del equipo de desarrollo con autonomía.
- Participar en retrospectivas y proponer mejoras al proceso de diseño.
- Comunicar de forma asertiva con stakeholders del proyecto.
