# Calidad y validación

Participa activamente en PDU. Aplica parámetros de accesibilidad en sus diseños. Solicita Design Review al referente antes de entregar.

**Ejemplos:**
- Preparar guías de prueba para sesiones de usabilidad.
- Validar accesibilidad de componentes con herramientas como Stark o Contrast Checker.
- Solicitar Design Review y aplicar feedback recibido de forma completa.
- Documentar resultados de PDU y proponer mejoras a partir de los hallazgos.
- Iterar sobre diseños hasta cumplir criterios de calidad acordados.
