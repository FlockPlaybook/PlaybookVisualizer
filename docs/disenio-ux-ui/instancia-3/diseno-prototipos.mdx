# Diseño y creación de prototipos

Crea interfaces de mediana complejidad para los productos tecnológicos desarrollados. Requiere menor acompañamiento del Líder de Diseño, logrando cumplir las tareas en tiempo y forma.

Cuenta con un conocimiento cada vez más creciente del uso de los patrones de diseño que lo habilitan a generar prototipados interactivos mediante las herramientas pertinentes.

**Ejemplos:**
- Considera algunos estados potenciales (error, éxito, callejones sin salida) en sus prototipos.
- Valida con desarrollo las decisiones de diseño que toma.
- Trabaja con styleguides y componentes provistos, creando diseños armónicos respetando las decisiones pautadas.
