# Diseño y creación de prototipos

Crea interfaces de mediana complejidad considerando estados potenciales del flujo (error, éxito, vacío). Valida sus diseños con el equipo de desarrollo.

**Ejemplos:**
- Diseñar flujos completos con manejo de estados de error y éxito.
- Proponer y validar soluciones de diseño con el desarrollador asignado.
- Iterar sobre prototipos de media fidelidad incorporando feedback del cliente.
- Aplicar el Design System de forma consistente en todas las pantallas.
- Documentar decisiones de diseño y justificaciones en el espacio de trabajo compartido.
