# Estrategia de producto

Colabora en el aterrizaje de los casos de uso y participa, en conjunto con el equipo, en el refinamiento de las tareas.

Tiene un conocimiento parcial de las necesidades y objetivos del proyecto.

**Ejemplos:**
- Puede transformar el input en flowmaps y sitemaps utilizando los templates y co-creando con el cliente.
- Participa en la redacción del documento de funciones junto a su equipo.
- Indaga sobre las decisiones tomadas para comprender mejor el producto.
