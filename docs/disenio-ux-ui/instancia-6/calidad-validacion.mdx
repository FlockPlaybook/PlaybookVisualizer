# Calidad y validación

Diseña contemplando escalabilidad. Analiza datos cualitativos y cuantitativos y genera reportes. Vela por la accesibilidad en etapas de QA y Desarrollo.

**Ejemplos:**
- Revisar componentes en desarrollo para asegurar fidelidad con los diseños.
- Generar reportes de accesibilidad y usabilidad para stakeholders externos.
- Establecer criterios de aceptación de calidad visual para el equipo de QA.
- Analizar datos de uso y proponer mejoras iterativas al producto.
- Liderar iniciativas de mejora de accesibilidad en toda la plataforma.
