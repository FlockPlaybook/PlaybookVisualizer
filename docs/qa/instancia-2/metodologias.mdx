# Metodologías y mejora continua

Completa tareas en tiempo y forma. Se ocupa de fechas clave del proyecto. Participa activamente en ceremonias ágiles.

**Ejemplos:**
- Gestionar el propio backlog de testing respetando los deadlines del sprint.
- Comunicar proactivamente cuando una tarea de testing puede impactar el timeline.
- Participar activamente en refinamientos aportando perspectiva de calidad.
- Trabajar el backlog de defectos con el referente de forma ordenada.
