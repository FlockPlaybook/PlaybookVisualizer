# QA - Instancia 2

## Descriptor

| Dimensión | Descripción |
|---|---|
| **Complejidad** | Problemas rutinarios con mayor variedad. Instrucciones detalladas o moderadas. |
| **Autonomía** | Requiere instrucciones moderadas. Algunas tareas con mayor autonomía. |
| **Impacto de las decisiones** | Las decisiones comienzan a impactar en el equipo. |
| **Alcance** | Contribución individual con creciente autonomía dentro del equipo. |
