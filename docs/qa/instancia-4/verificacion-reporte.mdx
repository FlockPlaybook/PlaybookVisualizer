# Verificación y reporte

Visión íntegra y holística del negocio en la verificación. Define acciones preventivas. Realiza consultas SQL complejas. Realiza pruebas de accesibilidad y/o SEO.

**Ejemplos:**
- Ejecutar pruebas con visión completa del negocio, anticipando impactos transversales.
- Definir acciones preventivas para mitigar riesgos de calidad antes de que ocurran.
- Realizar consultas SQL complejas para validar la integridad de datos en el backend.
- Ejecutar auditorías de accesibilidad (WCAG) y SEO técnico.
- Liderar la verificación de releases críticos con el equipo.
