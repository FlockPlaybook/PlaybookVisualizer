# Brinda un servicio de excelencia

Referente dentro del proyecto. Adapta el lenguaje al interlocutor. Propone backlog de tareas preventivo.

**Ejemplos:**
- Ser el punto de contacto de calidad dentro del proyecto.
- Adaptar la comunicación de métricas según el interlocutor (técnico, funcional, ejecutivo).
- Proponer un backlog de tareas preventivas para evitar defectos futuros.
- Facilitar la resolución de conflictos relacionados con calidad.
