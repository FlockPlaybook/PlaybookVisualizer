# Metodologías y mejora continua

Responsable de la calidad del proceso de desarrollo. Implementa indicadores de calidad. Comunica métricas del proyecto asociadas con QA.

**Ejemplos:**
- Implementar métricas de calidad (defect density, test coverage, escape rate).
- Comunicar el estado de calidad del proyecto a stakeholders internos.
- Proponer mejoras al proceso de desarrollo basadas en métricas de QA.
- Ser responsable del aseguramiento de calidad en todas las etapas del ciclo de desarrollo.
- Proponer un backlog de tareas preventivas de QA.
