# Verificación y reporte

Realiza pruebas en sistemas simples desde la interfaz gráfica. Su reporte de errores es claro y permite reproducir el defecto.

**Ejemplos:**
- Ejecutar casos de prueba manuales sobre la interfaz gráfica.
- Reportar defectos de forma clara, incluyendo pasos para reproducir, resultado esperado y actual.
- Verificar la corrección de defectos tras el fix del desarrollador.
- Documentar el resultado de cada ejecución de prueba.
