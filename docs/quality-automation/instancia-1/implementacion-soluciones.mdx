# Implementación de soluciones

Implementa soluciones parciales de una aplicación de dificultad media, cumpliendo con los atributos de calidad y los estándares de código definidos en los proyectos en los que participa.

**Ejemplos:**
- Cumple en tiempo y forma con sus tareas, incluso cuando depende de ayuda de terceros
- Propone al menos una solución posible para las tareas a su cargo
- Logra implementar soluciones a flujos y módulos de hasta complejidades media durante un mismo sprint.
- Realiza soluciones escalables y mantenibles dentro del proyecto.
- Durante el desarrollo de proyectos realiza implementaciones que puedan ser extraíbles y reutilizables.
- Consigue completar tareas tecnológicamente desafiantes en los tiempos estipulados.
- Implementa adecuadamente las verificaciones a flujos en escenarios normales, alternativos y erróneos.
- Maneja adecuadamente las variables de entorno.
- Ejecuta sus pruebas en distintos navegadores, dispositivos emuladores y/o físicos según corresponda
- Es capaz de gestionar la disponibilidad de la información o el set de datos requeridos para las pruebas.
- Sus soluciones se adecúan a los atributos de calidad definidos por la compañía (Funcionalidad, Compatibilidad, Mantenibilidad, Escalabilidad. Fiabilidad, Performance)
- Sus pruebas automatizadas son independientes y autónomas
- Construye código claro y documenta partes clave de las implementaciones que realiza.
- Selecciona e implementa adecuadamente los patrones de diseño del framework (page object, screenplay, etc.)
- Realiza pruebas considerando los tipos de usuarios o roles del sistema
- Genera la documentación necesaria para los entregables definidos
- Utiliza, con criterio y cuando es necesario, herramientas provistas por terceros, comprendiendo el impacto que las mismas tienen sobre la aplicación revisando:
- Frecuencia de actualización de un libreria que agregó o la confiabilidad
- Cantidad de stars en Github
- Peso (tamaño) que se le agrega a la aplicación
- Licencia actualizada o
- Variabilidad y/o consistencia con la modalidad de desarrollo del proyecto
