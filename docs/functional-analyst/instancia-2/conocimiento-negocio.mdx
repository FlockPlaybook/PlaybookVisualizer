# Conocimiento de negocio y estrategia de producto

Construye release plans. Comprende y define MVP. Construye roadmaps de producto. Realiza investigaciones de negocio y mercado.

**Ejemplos:**
- Elaborar release plans alineados con las necesidades del negocio.
- Definir el alcance del MVP y fundamentarlo con datos.
- Construir roadmaps de producto de corto y mediano plazo.
- Realizar investigaciones de mercado y competidores para informar decisiones de producto.
- Presentar hallazgos de negocio a stakeholders internos y externos.
