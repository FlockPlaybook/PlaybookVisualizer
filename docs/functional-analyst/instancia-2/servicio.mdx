# Brinda un servicio de excelencia

Referente técnico de análisis para el cliente. Se anticipa a pedidos. Gestiona conflictos de intereses. Realiza onboarding de nuevos miembros.

**Ejemplos:**
- Ser el punto de contacto principal para el cliente en temas de análisis funcional.
- Gestionar conflictos de intereses entre stakeholders de forma asertiva.
- Realizar onboarding de nuevos miembros del equipo.
- Anticiparse a cambios de alcance y comunicarlos proactivamente.
