# Metodologías y mejora continua

Define estrategia de análisis para el proyecto. Elabora minutas y genera canales de comunicación efectivos. Toma ownership de refinamientos.

**Ejemplos:**
- Definir la metodología de análisis más adecuada para cada proyecto.
- Liderar refinamientos y asegurar que los tickets estén bien definidos.
- Elaborar minutas de reuniones y distribuirlas oportunamente.
- Generar canales y protocolos de comunicación claros para el equipo.
- Proponer mejoras al proceso de trabajo basadas en retrospectivas.
