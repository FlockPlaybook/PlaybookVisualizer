# Functional Analyst - Instancia 3

## Descriptor

| Dimensión | Descripción |
|---|---|
| **Complejidad** | Crea nuevas soluciones ante problemas complejos de análisis frecuentemente. Mínima guía requerida. |
| **Autonomía** | Alta autonomía. Lidera procesos de análisis complejos de forma independiente. |
| **Impacto de las decisiones** | Las decisiones impactan en el equipo, en el cliente y ocasionalmente en otros equipos. |
| **Alcance** | Referente de disciplina para toda la compañía. Lidera estrategia del departamento. |
