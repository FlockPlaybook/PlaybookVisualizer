# Brinda un servicio de excelencia

Colabora proactivamente con otros equipos. Es referente técnico dentro y fuera del proyecto. Ofrece asesoramiento técnico a stakeholders.

**Ejemplos:**
- Colaborar proactivamente con equipos de producto, diseño y QA.
- Ser referente técnico para el cliente interno y externo.
- Ofrecer asesoramiento técnico a stakeholders no técnicos.
- Participar en reuniones comerciales aportando perspectiva técnica.
