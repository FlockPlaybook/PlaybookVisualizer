# Software Architect - Instancia 2

## Descriptor

| Dimensión | Descripción |
|---|---|
| **Complejidad** | Componentes de mediana y alta complejidad con mayor autonomía. |
| **Autonomía** | Alta autonomía. Decisiones sobre componentes de arquitectura con consultas al equipo. |
| **Impacto de las decisiones** | Las decisiones impactan en la arquitectura general del proyecto. |
| **Alcance** | Referente técnico en el proyecto. Guía al equipo de desarrollo. |
