# Calidad de los entregables

Mantiene altos estándares de calidad. Identifica fallas de diseño. Implementa pruebas unitarias y automatizadas. Supervisa la gestión de defectos.

**Ejemplos:**
- Implementar pruebas unitarias para los componentes que desarrolla.
- Identificar fallas de diseño y reportarlas al arquitecto senior.
- Participar en code reviews y aplicar feedback recibido.
- Supervisar la gestión de defectos en el área de su responsabilidad.
