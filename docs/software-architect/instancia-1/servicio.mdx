# Brinda un servicio de excelencia

Establece canales de comunicación claros. Asegura que las soluciones cumplan estándares de calidad. Es referente técnico eventual para el cliente.

**Ejemplos:**
- Establecer protocolos de comunicación claros con el equipo y el cliente.
- Asegurar que los entregables cumplan con los estándares de calidad definidos.
- Actuar como referente técnico ocasional ante preguntas del cliente.
