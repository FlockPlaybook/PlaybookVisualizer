# Metodología y mejora continua

Lidera iniciativas para optimizar procesos. Fomenta cultura de mejora continua. Utiliza KPIs para medir la efectividad de prácticas ágiles.

**Ejemplos:**
- Liderar iniciativas de mejora de procesos a nivel de proyecto.
- Fomentar una cultura de mejora continua e innovación en el equipo.
- Definir y monitorear KPIs de efectividad del proceso de desarrollo.
- Facilitar workshops de mejora de proceso con el equipo.
