# Brinda un servicio de excelencia

Comunica de manera clara y proactiva. Orienta al cliente como prioridad. Asiste al equipo comercial en reuniones y estimaciones.

**Ejemplos:**
- Liderar la comunicación técnica con el cliente de forma proactiva.
- Asistir al equipo comercial en reuniones de pre-venta y estimaciones.
- Resolver problemas del cliente de forma eficaz y eficiente.
- Construir relaciones de confianza de largo plazo con stakeholders clave.
