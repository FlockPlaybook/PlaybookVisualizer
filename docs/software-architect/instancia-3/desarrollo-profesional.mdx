# Desarrollo profesional

Actúa como mentor y líder técnico. Organiza charlas internas. Presenta en conferencias. Proporciona orientación individualizada a arquitectos en etapas iniciales.

**Ejemplos:**
- Organizar tech talks y charlas internas sobre arquitectura y tecnología.
- Presentar en conferencias externas representando a la organización.
- Proporcionar mentoría profunda a arquitectos junior y mid.
- Definir el programa de desarrollo técnico del área.
