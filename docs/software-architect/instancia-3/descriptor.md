# Software Architect - Instancia 3

## Descriptor

| Dimensión | Descripción |
|---|---|
| **Complejidad** | Capacidad para abordar tareas complejas y desafiantes. Alta autonomía. |
| **Autonomía** | Alta autonomía. Lidera decisiones técnicas de alto impacto. |
| **Impacto de las decisiones** | Impacto significativo en la arquitectura del proyecto y los resultados del equipo. |
| **Alcance** | Líder técnico. Referente de arquitectura para el proyecto y el área. |
