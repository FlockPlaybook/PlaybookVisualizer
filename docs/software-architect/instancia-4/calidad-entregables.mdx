# Calidad de los entregables

Lidera la definición de estándares de calidad extremadamente elevados. Gestión proactiva de defectos. Responsable de mejora continua de métricas de calidad.

**Ejemplos:**
- Definir e implementar estándares de calidad técnica para toda la organización.
- Liderar revisiones de arquitectura y código a nivel organizacional.
- Gestionar proactivamente defectos críticos de producción.
- Planificar y ejecutar pruebas avanzadas de rendimiento y seguridad.
- Ser responsable de la mejora continua de las métricas de calidad técnica.
