# Software Architect - Instancia 4

## Descriptor

| Dimensión | Descripción |
|---|---|
| **Complejidad** | Identifica y evalúa problemas complejos con visión holística. |
| **Autonomía** | Máxima autonomía. Visión estratégica de la arquitectura de la organización. |
| **Impacto de las decisiones** | Gran impacto en el área de responsabilidad y en la organización. |
| **Alcance** | Referente de arquitectura para toda la compañía. Define la dirección técnica. |
