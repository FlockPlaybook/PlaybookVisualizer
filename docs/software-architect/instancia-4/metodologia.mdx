# Metodología y mejora continua

Lidera definición y aplicación de metodologías avanzadas. Automatiza el proceso de CI/CD. Implementa monitoreo continuo y prácticas de rollback.

**Ejemplos:**
- Definir la metodología de desarrollo de la organización.
- Liderar la automatización completa del pipeline de CI/CD.
- Implementar prácticas de observabilidad, monitoreo y rollback.
- Gestionar proyectos de arquitectura complejos de forma autónoma.
